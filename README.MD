# 安装与使用

## 安装谷歌浏览器

从各个应用商店安装即可

## 安装python环境

建议使用Anaconda进行版本管理
(Anaconda安装教程)[http://newgr8player.gitee.io/2018/03/24/Python%E9%9B%86%E6%88%90%E7%8E%AF%E5%A2%83Anaconda%E4%BD%BF%E7%94%A8/]

## 将项目下载到本地

可以选择下载压缩包或者
```
git clone https://github.com/NewGr8Player/philosophical_flavour.git
```

## 使用Anaconda安装所需要的包
```
requests
bs4
openpyxl
selenium
```

## 运行程序
导入IDE或者使用命令
```
python Main.py
```
